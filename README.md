- **Bias Detection:** Real-time monitoring for anchoring, overconfidence, and circular reasoning within the simulation.
- **Collapse Detection:** Automatic monitoring of system coherence and entropy spikes.

### Layer 2: Cross-Domain Discovery (The "Universe")
- **Universal Adapters (`cross_domain/`):** Application of emergent spacetime laws to:
    - **Finance:** Volatility tracking and regime detection.
    - **Genomics:** SNP indexing and variant classification.

    - **Drug Discovery:** Compound-target activity prediction.
    - **NLP:** Quantum-inspired POS tagging via Viterbi decoding.

## 🚀 Getting Started

### Prerequisites
- Python 3.12+
- Dependencies: `numpy`, `scipy`, `scikit-learn`

### Running the System
The entire RCF pipeline (Study → Understand → Integrate → Test → Validate → Discover) can be executed via the main entry point:

```bash
PYTHONPATH=. python3 main.py
```

This will generate a comprehensive `system_report.json` containing all performance metrics, Q-scores, and cross-domain findings.

### Running Tests
To verify all subsystems independently:

```bash
PYTHONPATH=. python3 test_all.py
```
